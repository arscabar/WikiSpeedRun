# WikiSpeedRun v0.1.3

## 핵심 변경

- 자동 제시어를 실제 본문 링크로 도달 가능한 문서만 나오도록 변경했습니다.
- 앱 화면에 `도달 검증: n클릭 경로 존재` 표시를 추가했습니다.
- Cloudflare Tunnel 외부 접속 주소를 앱 안에서 자동 표시하고 복사할 수 있게 했습니다.
- 앱/서버/터널을 새로 시작할 때 이전 WikiSpeedRun 서버와 같은 포트의 터널을 정리합니다.
- 릴리즈 ZIP 하나만 받아 압축을 풀고 `Start-WikiSpeedRun-Cloudflare.cmd`를 실행하면 앱과 외부 공유를 시작할 수 있습니다.

## 포함 파일

- `WikiSpeedRun-0.1.3-portable.exe`
- `cloudflared.exe`
- `Start-WikiSpeedRun-Cloudflare.cmd`
- `Publish-WikiSpeedRun-Tunnel.ps1`
- `README-cloudflare.txt`
