# WikiSpeedRun v0.1.4

## 핵심 변경

- 자동 제시어를 시작어/목표어 독립 랜덤으로 변경했습니다.
- 목표어는 나무위키 역링크 `(link)` 유입이 있는 문서만 사용해, 고립 문서가 목표로 나오는 문제를 줄였습니다.
- 플레이어 화면에서 힌트성 `n클릭 경로 존재` 표기를 제거했습니다.
- 인게임 문서에 역링크 가로 레일을 추가하고, 역링크 클릭 이동을 서버가 검증하도록 했습니다.
- 이동 기록과 결과 그래프에서 역링크 이동을 `역링크`로 구분합니다.
- 나무위키 목차를 `1. 개요`, `5.1. 큰 틀`처럼 구조를 유지해 표시합니다.
- 모바일 360/390/414px에서 대기실, 플레이, 찾기, 결과 화면 가로 overflow를 재검토했습니다.
